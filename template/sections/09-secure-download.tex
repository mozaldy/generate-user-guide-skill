\section{Secure Download}

\begin{ugModule}{Overview}

  \textbf{Purpose:}
  Provide tamper-proof document downloads with rasterization,
  embedded watermarking, and rate limiting.

  \textbf{Who Can Access:}
  \ugRole{Admin} \ugRole{Manager} \ugRole{Reviewer} \ugRole{Staff}
\end{ugModule}

\subsection{How It Works}

When a document is downloaded, the following processing pipeline
executes:

\begin{ugSteps}
  \item Each page is \textbf{rasterized} --- converted from
        vector PDF to a high-resolution image.
  \item A \textbf{watermark} is embedded on every page
        containing:
        \begin{itemize}[leftmargin=14pt]
          \item The downloader's full name
          \item Their IP address
          \item Download timestamp
          \item Browser / user agent
        \end{itemize}
  \item The images are reassembled into a new PDF.
  \item The resulting file is delivered to the user.
\end{ugSteps}

\begin{ugImportant}
  The downloaded PDF contains \textbf{image-based pages only}.
  Text cannot be selected, copied, or extracted. The watermark
  is permanently embedded in the image and cannot be removed.
\end{ugImportant}

\subsection{Rate Limiting}

Downloads are rate-limited to prevent abuse:

\begin{tabularx}{\textwidth}{@{} l X @{}}
  \toprule
  \textbf{Parameter} & \textbf{Value} \\
  \midrule
  Limit Scope     & Per user, per document, per hour \\
  Default Limit   & 10 downloads \\
  Reset Window    & Rolling 1-hour window \\
  Exceeded Action & HTTP 400 error returned \\
  Configuration   & Adjustable by administrator \\
  \bottomrule
\end{tabularx}

\subsection{Download Cache}

Rasterized results are cached per user per document version.
Subsequent downloads of the same version by the same user
are served from cache for optimal performance, avoiding
repeated rasterization processing.

\subsection{Step-by-Step: Downloading a Document}
\begin{ugSteps}
  \item Navigate to the document detail page.
  \item Click \ugButton{Download} for the latest version.
  \item To download a specific version, select from the
        \ugField{Version} dropdown first.
  \item Wait for rasterization and watermark processing
        (first-time downloads may take longer).
  \item The file downloads automatically to your browser's
        default download folder.
\end{ugSteps}

\begin{ugTip}
  First-time downloads require full rasterization processing.
  Subsequent downloads of the same document version are served
  from cache and will be significantly faster.
\end{ugTip}
